## 🔧 Customization

### To Change Fees:
```sql
-- Update displayed fee
UPDATE delivery_cities 
SET displayed_fee = 3.00 
WHERE city_name_en = 'Amman';

-- Update actual fee  
UPDATE delivery_cities 
SET actual_fee = 3.50 
WHERE city_name_en = 'Aqaba';
```

### To Add More Cities:
```sql
INSERT INTO delivery_cities 
(country_id, city_name_en, city_name_ar, actual_fee, displayed_fee, is_active) 
VALUES 
(1, 'New City', 'مدينة جديدة', 2.50, 2.00, TRUE);
```

### To Deactivate Country/City:
```sql
-- Deactivate city
UPDATE delivery_cities SET is_active = FALSE WHERE id = 5;

-- Deactivate country
UPDATE delivery_countries SET is_active = FALSE WHERE id = 2;
```
